# 📅 Calendar & Reminder App — Desktop Version

A full-featured Python tkinter desktop application.

## 🚀 How to Run

1. Install dependencies:
```bash
pip install tkcalendar plyer requests
```

2. Run the app:
```bash
python main.py
```

## ✨ Features
- 📅 Interactive calendar with reminders
- 🔴🟡🟢 Priority levels (High/Medium/Low)
- 😊 Mood tracker with journal
- 🎂 Birthday manager with alerts
- 🌤️ Live weather in footer
- 🔔 Sound notifications
- 💬 Quote of the day
- 🔍 Search reminders
- 📤 Export to CSV
- ☀️🌙 Dark/Light theme
- ⏳ Countdown to next reminder
- 💡 Splash screen

## 📁 Files
- `main.py` — Entry point
- `app.py` — Main application window
- `calendar_view.py` — Calendar widget
- `reminder_manager.py` — Reminders panel
- `mood_tracker.py` — Mood tracker
- `birthday.py` — Birthday manager
- `storage.py` — Data storage
- `notifier.py` — Background notifications
- `quotes.py` — Daily quotes
- `countdown.py` — Countdown widget
- `weather_widget.py` — Weather display
- `splash.py` — Splash screen
